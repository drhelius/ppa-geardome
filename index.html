APT Repository for DrHelius Emulators

This is the official APT repository (PPA) for DrHelius emulators, served via GitHub Pages.

Available Emulators

  • Gearboy — Game Boy / Game Boy Color / Super Game Boy
  • Gearsystem — Sega Master System / Game Gear / SG-1000
  • Gearcoleco — ColecoVision
  • Geargrafx — TurboGrafx-16 / PC Engine / SuperGrafx / CD-ROM²
  • Gearlynx — Atari Lynx

See the repository README for installation instructions.
